Squarespace embed snippet

Paste this iframe into a Squarespace Embed Block. Update src to your host.

<iframe src="https://tools.digitallyfestyle.com/nomad-budget-calculator/" width="100%" style="min-height:760px;border:0;" allow="clipboard-read; clipboard-write" loading="lazy"></iframe>

Optional auto-height

Include this script on the Squarespace page to smoothly adapt the iframe height. Replace ALLOWED_ORIGIN with your calculator origin (e.g. https://tools.digitallyfestyle.com).

<script>
const iframe = document.querySelector('iframe[src*="nomad-budget-calculator"]');
const ALLOWED_ORIGIN = 'https://tools.digitallyfestyle.com';
window.addEventListener('message', (event) => {
  if (!iframe) return;
  if (event.origin !== ALLOWED_ORIGIN) return;
  if (event.data?.type === 'nbc-height' && Number.isFinite(event.data.height)) {
    iframe.style.height = Math.max(event.data.height, 760) + 'px';
  }
});
</script>
